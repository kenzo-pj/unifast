High-performance Markdown/MDX compiler with an Oxc-style architecture.

## Features

- **Fast** - Rust core with zero-copy parsing
- **Complete** - CommonMark, GFM, frontmatter, MDX
- **Safe** - Built-in HTML sanitization
- **Portable** - Node.js (N-API), WASM, CLI
