## Overview

`@unifast/plugin-frontmatter` enables frontmatter parsing in your Markdown/MDX documents. It supports YAML (enabled by default), TOML, and JSON frontmatter formats. Parsed metadata is available in the `frontmatter` field of the compile result.

## Installation

<PackageInstall package="@unifast/plugin-frontmatter" />

## Type Definitions

### `FrontmatterPluginOptions`

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `yaml` | `boolean` | `true` | Enable YAML frontmatter (`---` delimiters) |
| `toml` | `boolean` | `false` | Enable TOML frontmatter (`+++` delimiters) |
| `json` | `boolean` | `false` | Enable JSON frontmatter (opening `{`) |

### Produced `CompileOptions`

When applied, this plugin merges the following into `CompileOptions`:

```ts
{
  frontmatter: {
    yaml: boolean,
    toml: boolean,
    json: boolean,
  }
}
```
