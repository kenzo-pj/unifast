## Overview

`@unifast/plugin-toc` enables automatic extraction of a table of contents from headings in your Markdown/MDX documents. The extracted TOC entries are available in the `toc` field of the compile result, each containing the heading depth, text, and generated slug.

## Installation

<PackageInstall package="@unifast/plugin-toc" />

## Type Definitions

### `TocPluginOptions`

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `maxDepth` | `number` | `3` | Maximum heading depth to include (1-6). A value of `3` includes h1, h2, and h3. |

### `TocEntry`

Each entry in `CompileResult.toc`:

| Property | Type | Description |
|----------|------|-------------|
| `depth` | `number` | Heading level (1 for `#`, 2 for `##`, etc.) |
| `text` | `string` | Plain text content of the heading |
| `slug` | `string` | Auto-generated slug for anchor linking |

### Produced `CompileOptions`

```ts
{
  toc: {
    enabled: true,
    maxDepth: number,
  }
}
```

## Exports Summary

| Export | Kind | Description |
